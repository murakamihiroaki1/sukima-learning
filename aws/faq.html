FAQ - Frequently Asked Questions

サービスについて

このサービスは何ですか？
AWS認定試験の合格を目指す方のための学習支援サービスです。実践的な問題演習と詳細な解説で、効率的な試験対策をサポートします。
どの認定試験に対応していますか？
現在、以下の認定試験に対応しています：
  • Foundational 認定
    • CLF - AWS Certified Cloud Practitioner (CLF-C02)
    • AIF - AWS Certified AI Practitioner (AIF-C01)
  • Associate 認定
    • SAA - AWS Certified Solutions Architect - Associate (SAA-C03)
    • MLA - AWS Certified Machine Learning Engineer - Associate (MLA-C01) ※近日公開
    • SOA - AWS Certified SysOps Administrator - Associate (SOA-C02) ※近日公開
    • DEA - AWS Certified Data Engineer - Associate (DEA-C01) ※近日公開
    • DVA - AWS Certified Developer - Associate (DVA-C02) ※近日公開
  • Professional 認定
    • 今後対応予定

料金・プランについて

無料プランでどこまで利用できますか？
無料プランでは、サンプル問題（20問）にアクセスできます。本格的な学習には有料プランをおすすめします。
支払い方法は何がありますか？
クレジットカード（Visa、Mastercard、JCB、American Express）に対応予定です。

学習機能について

カテゴリ別学習とは何ですか？
コンピューティング、ストレージ、データベースなど、AWSサービスのカテゴリごとに問題を絞り込んで学習できる機能です（Standardプラン以上）。
模擬試験モードとは何ですか？
実際の試験と同じ形式・時間制限で問題に取り組める機能です。本番前の実力確認に最適です（Standardプラン以上）。

技術的な質問

スマートフォンでも利用できますか？
はい、スマートフォン、タブレット、PCなど、あらゆるデバイスでご利用いただけます。レスポンシブデザインで最適な表示を提供します。
