## 購入手順（Cloudflare Registrar の例）

### ステップ1: Cloudflareアカウント作成
1. https://dash.cloudflare.com/sign-up にアクセス
2. メールアドレスとパスワードを入力
3. メール認証を完了

### ステップ2: サイトを追加
1. ダッシュボードで「サイトを追加」をクリック
2. 購入したいドメイン名を入力（例: example.com）
3. Freeプランを選択

### ステップ3: ドメインを購入
1. 左メニューから「ドメイン登録」を選択
2. 希望のドメイン名を検索
3. 「購入」をクリック
4. 支払い情報を入力（クレジットカード）
5. 購入完了

### ステップ4: DNS設定
1. 「DNS」タブを開く
2. 必要なレコードを追加（後述）

## DNS設定

### Vercel/Netlifyにデプロイする場合

#### Aレコード（IPv4）
```
Type: A
Name: @
Content: <Vercel/NetlifyのIPアドレス>
TTL: Auto
```

#### CNAMEレコード（www）
```
Type: CNAME
Name: www
Content: <your-project>.vercel.app
TTL: Auto
```

### Vercelの場合の具体例
```
Type: A
Name: @
Content: 76.76.21.21
TTL: Auto

Type: CNAME
Name: www
Content: cname.vercel-dns.com
TTL: Auto
```

### Netlifyの場合の具体例
```
Type: A
Name: @
Content: 75.2.60.5
TTL: Auto

Type: CNAME
Name: www
Content: <your-site>.netlify.app
TTL: Auto
```

### Cloudflare Email Routing 設定手順（無料）

#### ステップ1: Email Routingを有効化
1. Cloudflareダッシュボードにログイン
2. 対象ドメインを選択
3. 左メニューから「Email」→「Email Routing」を選択
4. 「Enable Email Routing」をクリック

#### ステップ2: 転送先アドレスを追加
1. 「Destination addresses」セクションで「Add」をクリック
2. 転送先のメールアドレスを入力（例: your-gmail@gmail.com）
3. 確認メールが届くので、リンクをクリックして認証

#### ステップ3: ルーティングルールを作成
1. 「Routing rules」セクションで「Create address」をクリック
2. カスタムアドレスを入力（例: info@yourdomain.com）
3. 転送先を選択
4. 「Save」をクリック

#### ステップ4: DNS設定（自動）
Cloudflareが自動的に以下のレコードを追加します：
```
Type: MX
Name: @
Content: isaac.mx.cloudflare.net
Priority: 1

Type: TXT
Name: @
Content: v=spf1 include:_spf.mx.cloudflare.net ~all
```

#### 完了
- info@yourdomain.com 宛のメールが your-gmail@gmail.com に転送されます
- 複数のアドレスを無制限に作成可能

### ImprovMX 設定手順（無料）

#### ステップ1: アカウント作成
1. https://improvmx.com/ にアクセス
2. 「Get Started」をクリック
3. メールアドレスとパスワードを入力

#### ステップ2: ドメインを追加
1. ダッシュボードで「Add Domain」をクリック
2. ドメイン名を入力（例: yourdomain.com）
3. 転送先メールアドレスを入力

#### ステップ3: DNS設定
DNSプロバイダー（Cloudflare等）で以下のレコードを追加：

```
Type: MX
Name: @
Content: mx1.improvmx.com
Priority: 10

Type: MX
Name: @
Content: mx2.improvmx.com
Priority: 20

Type: TXT
Name: @
Content: v=spf1 include:spf.improvmx.com ~all
```

#### ステップ4: 確認
1. ImprovMXダッシュボードで「Verify」をクリック
2. DNS設定が反映されるまで待つ（最大48時間）

## コスト削減のコツ

### 1. **初年度割引を活用**
- Namecheap、お名前.comなどは初年度が激安
- ただし更新料金を確認すること

### 2. **複数年契約**
- 2-3年契約で割引がある場合も
- ただし、サービスを変更できなくなる

### 3. **Cloudflare Registrarに移管**
- 他社で購入後、Cloudflareに移管すると原価のみに
- 移管手数料は1年分の更新料金

### 4. **不要なオプションを外す**
- Whois保護: Cloudflare、Namecheap、Porkbunは無料
- SSL証明書: Let's Encryptで無料
- メール転送: 不要なら外す

### 5. **プロモーションコードを探す**
- RetailMeNot、Honey、クーポンサイトをチェック
- 初回購入で10-20%オフのコードがある

### 6. **比較サイトを活用**
- https://tld-list.com/ でTLD別の最安値を比較
- 更新料金も含めて比較
